# DataSets

This folder mirrors the datasets used by Aurelius for manual export, upload, and end-to-end testing.

## Layout
- `raw/` - source datasets and reference files
- `processed/` - cleaned CSVs used by the app import flow
- `templates/` - CSV templates for manual uploads

## Use
- Upload files from `processed/` or `templates/` to test the import flow.
- Use `raw/` for source/reference data when you want to inspect the original files.
